# Pinecone Setup Guide for JARVIS

## Step 1: Create Pinecone Account

1. **Visit Pinecone Website**
   - Go to: https://www.pinecone.io/

2. **Sign Up**
   - Click "Sign Up" or "Get Started Free"
   - You can sign up with:
     - Google account (fastest)
     - GitHub account
     - Email address

3. **Verify Email**
   - Check your email for verification link
   - Click to verify your account

## Step 2: Get Your API Key

1. **Log in to Pinecone Console**
   - Go to: https://app.pinecone.io/

2. **Navigate to API Keys**
   - Look for "API Keys" in the left sidebar
   - OR go directly to the dashboard

3. **Create an API Key**
   - Click "Create API Key" button
   - Give it a name like "jarvis-assistant"
   - Click "Create Key"

4. **Copy Your API Key**
   - ⚠️ IMPORTANT: Copy the key immediately!
   - It looks like: `pcsk-xxxxx-xxxxxxxxxxxxxxxxxxxxxxxxxx`
   - You won't be able to see it again after closing the dialog

## Step 3: Find Your Environment/Region

When you create your account, Pinecone assigns you a region.

Common regions:
- `us-east-1` (US East)
- `us-west-2` (US West)
- `eu-west-1` (Europe)
- `asia-southeast-1` (Asia)

You can find this in your Pinecone console dashboard.
